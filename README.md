# project-boost
First of three games taught on GameDev.tv's Udemy course. Starting to get the hang of GitHub, also now the game has a gameplay loop, detects collisions and restarts/quits when certain things are done!
How to play: 
- Press W/Space/UP - go upwards;
- Press A/Left - tilt it to the left;
- press D/right - tilt it to the right;

Any feedback is appreciated!
 
Current Version: 0.35
